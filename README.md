# Photo PDF Maker

画像を並べ替えてPDFにまとめる、依存関係なしの小さなブラウザアプリです。

公開URL: https://meta1529-dev.github.io/photo-pdf-maker/

教科書や書類をスマホアプリでスキャンしてJPG/PNGとして書き出したあと、別アプリで結合してPDF化する手間を減らすために作りました。

## できること

- 複数画像の追加
- ドラッグやボタンでの並べ替え
- 画像ごとの左右回転
- A4縦/A4横/A4自動/写真比率の切り替え
- 余白、配置、画質、解像度の調整
- PDFファイルの作成

## 使い方

`index.html` をブラウザで開きます。

1. `画像を追加` から画像を選ぶ
2. 必要なら順番や回転を調整する
3. ファイル名やページ設定を整える
4. `PDFを作成` を押す

画像はサーバーに送信されず、ブラウザ内でPDF化されます。

## ファイル構成

- `index.html`: 画面
- `styles.css`: 見た目
- `app.js`: 画像処理とPDF生成
